首先打开 DeepL 官网，点击:

```
Log in
```

```
Sign up
```

输入邮箱，点击:

```
Contiune
```

输入密码，并勾选服务条款，点击:

```
Create account
```

然后前往邮箱，点击 DeepL 邮箱验证链接即可完成注册。

现在我们拥有的是普通 DeepL 账号，需要升级到 DeepL API 账号。

点击左上角:

```
Products
```

```
DeepL API
```

点击:

```
See pricing
```

```
Sign up for free
```

注册免费的 DeepL API 账号。接着需要填写账单地址和银行卡（外币银行卡）信息来验证用户真实身份，点击:

```
Begin subscription
```

回到 DeepL 首页，点击右上角账号头像，点击:

```
Account
```

```
API Key & Limits
```

```
Create key
```

Name your key (optical) 随便命名，点击:

```
Create key
```

现在就有一个 DeepL API 了。
